# six

> Set Interval X

Limit the amount of times that `setInterval` will fire.

## Usage

This packages works on Deno, Node.js, and Browser.

### Deno

```typescript
import { setIntervalX } from "https://deno.land/x/six/mod.ts";
```

### Node

```javascript
import { setIntervalX } from "@ultirequiem/six";
```
